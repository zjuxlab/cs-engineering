什么是 LaTeX

概要​

LaTeX (发音为“ LAY -tek”或“ LAH -tek”) 是一种基于 TeX 的排版系统，由图灵奖得主 Lamport 开发，而 Tex 则是由 Knuth 最初开发。LaTeX 和常见的所见即所得的 Word 文档最大的区别就是用户只需要关注写作的内容，而排版则完全交给软件自动完成。

优势​

  • 支持为物理科学排版极其复杂的数学、表格和技术内容；
  • 脚注、交叉引用和书目管理的设施；
  • 易于生成复杂或乏味的文档元素，例如索引、词汇表、目录、图表列表；
  • 由于其固有的可编程性和可通过数千个免费附加包进行扩展，因此可高度定制用于定制文档制作。
