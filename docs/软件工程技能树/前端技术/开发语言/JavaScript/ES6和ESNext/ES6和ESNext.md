# ES6和ES Next

<b>📌 学习路线：</b>

1. ES6核心新特性
    1. let/const、模板字面量
    2. 箭头函数、解构赋值、展开运算符
    3. 类与继承（class）

2. 模块化（ES Module）
3. Promise 与异步编程
4. async/await 异步语法糖
5. 新数据结构（Map、Set、WeakMap、WeakSet）
6. 高级语法（可选链 ?.、空值合并运算符 ??、BigInt等）
