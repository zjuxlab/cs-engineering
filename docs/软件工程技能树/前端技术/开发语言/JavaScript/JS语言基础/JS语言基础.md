# JS语言基础

<b>📌 学习路线：</b>

1. JavaScript简介与环境搭建
2. 基础语法（变量、表达式、流程控制）
3. 数据类型（原始类型、引用类型）
4. 函数与作用域（函数定义、作用域链、闭包）
5. 对象与数组（定义与操作方法）
6. 基础DOM与事件处理（操作HTML元素、事件监听）
