# Angular

Angular 是由谷歌开发的一款基于TypeScript的开源前端框架，用于构建复杂、动态的单页应用程序（Single Page Application, SPA）。

Angular 提供了完整的前端解决方案，包括组件化、依赖注入、路由、表单处理、HTTP通信等功能，适用于中大型前端项目，尤其是企业级应用开发。
