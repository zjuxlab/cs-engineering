transition

Author:徐旻昶

transition动画简介

transition意译为中文应该叫做“过渡”，用比较科学的话来描述这一动画效果即为“将某一属性的开始值在线性时间内变化到结束值”。

如果您还对transition的效果了解并不是很直观，可以观看下面这张动图的效果，当鼠标移动到这个图标上时，这个图标在线性时间内被均匀的放大了，这就是transition的基础效果。

transition的四个属性

在了解transition的具体属性之前，我们先来看看上面预览图的简单过渡效果是如何实现的

<img ... class="image">   /*显示图片的html代码*/
...
...
.image{
  ...
  width: 100px;   /*正常情况下效果*/
  transition: width 3s;   /*transition属性*/
}
.image:hover{
  width: 300px;   /*鼠标覆盖到图片上时的效果*/
}

在鼠标未移动到图片上时，图片采用的class是image，它的宽度为100px；而鼠标移动到图片上时采用image:hover的class，宽度为300px；而实现100px到300px的线性过渡效果的，就是这一句

transition: width 3s;

看起来非常方便是吗（

接下来我们将从每个transition属性的参数入手来了解过渡语句的含义

transition-property​

transition-property用于设置元素中参与过渡的属性都有哪些，您可以简单的把它理解为一个要加入transition活动的属性名单

transition-property: none | all | property;
  1. 如果您将property设置为none，则表明没有任何属性会被这一条过渡属性影响
  2. 如果您将property设置为all，则表明该元素的所有属性都会参与到过渡动画当中
  3. 如果您将property设置为详细的属性名，则被您列出的属性名会参与到过渡动画当中，不同的属性之间用逗号分隔。

transition-duration​

transition-duration用于设置过渡动画效果持续的时间，直接设置为单位为s或ms的数字即可

transition-duration：.3s， 20ms；
  1. 如果您将duration设置为0，则该过渡效果不会生效
  2. 您可以省略小数点前的0
  3. 当您的property有用逗号隔开的多个属性时，duration可以为每一个属性设置对应的持续时间，如上述代码块，也用逗号隔开一一对应

transition-timing-function​

transition-timing-function用于设置过渡动画的类型，可选属性有：

value

description

linear

以相同的速度完成过渡变化，线性时间变化

ease

开始时慢速，中间较快，结束时慢速

ease-in

开始时慢速，后来变快

ease-out

开始时快速，后来变慢

ease-in-out

开始时慢速，中间加速到一个固定速度保持一段时间，结束时慢速

cubic-bezier(n, n, n, n)

自定义过渡速度，每个参数的范围都是0-1

transition-timing-function: cubic-bezier(.25, .1, .25, 1.0);

  1. cubic-bezier意为贝塞尔曲线，这是一条控制过渡速度变化的曲线，只需要有四个点就可以自动进行该曲线的生成。曲线每一处的斜率对应了每一时刻的过渡动画速率（因为横纵轴都是0-1，所以您将哪个轴理解成时间轴都没关系），函数中的4个n对应的分别是（x1，y1，x2，y2），即P1、P2两点的坐标，而P0和P3在transition中是固定不变的。我们可以通过设置P1、P2的坐标值完成任何速率的过度变化，如ease变化等效于（0.25，0.1，0.25，1.0）。
  2. 可能有弹幕会问了，哎呀ease和ease-in-out不都是慢快慢吗，没什么性质上的区别也要占两个关键字？别急，ease-in-out由慢速会在大概1/3的时间变为快速，然后保持在这个速度一段时间，再变为慢速；ease由慢速变为快速会一直增长，增长到最快速时再渐渐减慢速度。在贝塞尔曲线上反应，ease等效于cubic-bezier(0.25, 0.1, 0.25, 1.0)，而ease-in-out等效于cubic-bezier(0.42, 0, 0.58, 1.0)，我们可以在图中明显看到不同。

transition-delay​

transition-delay用于设置过渡动画在执行前需要等待的时间，数据类型同duration

transition-delay: .3s，20ms;
  1. 当数值为负时，过渡效果会直接开始
  2. 多个delay时间的使用同duration，与property一一对应，逗号隔开

transition同时对四个属性进行设置​

如可以用以下例子：

transition:property1 duration1 function1 delay1, property1 duration2...

将上述四个属性的值区分开，每个property用逗号隔开，每个property对应的四个属性值用空格隔开，可以缺省。

transition的使用

可能大家会有疑问，transition是两个状态之间的变化，怎么对一个元素写两个class的属性让其“动起来”呢？

css很贴心地为我们提供了鼠标事件，使用div:mouseMove就可以编写“当鼠标怎么怎么样”时的状态了

事件

代码

div

假设有一个分块叫div，我们使用div设置它的style

div:hover

当鼠标悬停在这个div上时的style

div:link

在鼠标点击这个div前的style

div:active

在鼠标点击这个div时的style

div:visited

在鼠标点击这个div后的style

div:focus

鼠标聚焦时的style（如输入框）

当然，如果您想让过渡动画运用到一些比较复杂的鼠标响应事件当中，可能您会需要VS的知识。

其实transition动画本身是让一个html的元素在两种css style之间进行过度转化的过程，所以您也可以这么写：

/*这里是html伪代码*/
<if flag==0>  <div class="style1"> ...
<else-if flag==1>  <div class="style2">...

/*这里是css伪代码*/
style1 {
    transition:...}
style2 {
    transition:...}

同样可以达到过渡动画的效果
