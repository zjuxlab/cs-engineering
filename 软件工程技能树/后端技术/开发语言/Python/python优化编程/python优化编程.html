常见范式：让别人见了直呼Pythonic

  • 字符串的处理：输入(bytes->str)→处理(str)→输出(str->bytes)

    • 也就是除了IO部分一律使用str可以减少很多事

  • 多用f-string，而不是format等经典方法：f-string格式化的可读性好得多，且冗余代码较少

  • 不要吝啬在函数内部定义辅助函数：到处用lambda表达式和if-else三元表达式并不会显得很帅

  • 可以试试恰当运用赋值表达式（海象运算符:=）来简化代码

  • 用enumerate去遍历，而非range取下标i再访问a[i]

  • 熟练运用Unpacking，尤其是单星号和双星号- 复习：想想*args和**kwargs是什么意思？ - 个人案例： ![](/assets/Eq7HbUaEIo2JXExwPvbcXRStnIb.png)
  • 不要用for-else和while-else结构：我觉得这是python设计的一个败笔- 这带来了语义上的歧义 - 看看隔壁Rust能break出去一个值，这是好的
