回归

Author: 葛佳惠

一元线性回归​

  • 对于一组n个测量(XiX_iXi​,YiY_iYi​)值，认为Y与X符合线性关系

Yi=b0+b1Xi+eiY_i=b_0+b_1X_i+e_iYi​=b0​+b1​Xi​+ei​

  • b0b_0b0​截距，b1b_1b1​斜率
  • Y的方差为: var(Y)=var(e)=var(Y) = var(e) =var(Y)=var(e)=σ2\sigma^2σ2
  • 如何估计b0b_0b0​和b1b_1b1​?

最小二乘法（Least Squares）​

希望Y的预测值与测量值偏差越小越好,对于n个测量

Q=∑(Yi−(b0+b1Xi))2Q=\sum(Y_i−(b_0+b_1X_i))^2Q=∑(Yi​−(b0​+b1​Xi​))2

最小化Q,最小二乘法

将回归函数的表达式改写为

Y=β0+β1(X−Xˉ)Y=β_0+β_1(X−\bar{X})Y=β0​+β1​(X−Xˉ)

上式被称为模型的中心化

Q=∑[Yi−(β0+β1(Xi−Xˉ))]2Q=\sum[Y_i−(β_0+β_1(X_i−\bar{X}))]^2Q=∑[Yi​−(β0​+β1​(Xi​−Xˉ))]2

对β0β_0β0​和β1β_1β1​求偏导

∂Q∂β0=−2∑[Yi−β0−β1(Xi−Xˉ)]=0\frac{\partial Q}{\partial β_0}=−2\sum[Y_i−β_0−β_1(X_i−\bar{X})]=0∂β0​∂Q​=−2∑[Yi​−β0​−β1​(Xi​−Xˉ)]=0

∂Q∂β1=−2∑[Yi−β0−β1(Xi−Xˉ)](Xi−Xˉ)=0\frac{∂Q}{∂β_1}=−2∑[Y_i−β_0−β_1(Xi−\bar{X})](X_i−\bar{X})=0∂β1​∂Q​=−2∑[Yi​−β0​−β1​(Xi−Xˉ)](Xi​−Xˉ)=0

得到

β0^=∑Yi/n=Yˉ\hat{β_0}=∑Y_i/n=\bar{Y}β0​^​=∑Yi​/n=Yˉ

β1^=∑(Xi−Xˉ)YiSX2\hat{β_1}=\frac{∑(X_i−\bar{X})Y_i}{{S_X}^2}β1​^​=SX​2∑(Xi​−Xˉ)Yi​​

β^0\hatβ_0β^​0​,β^1\hatβ_1β^​1​均为YiY_iYi​的线性函数

  • β^0\hatβ_0β^​0​,β^1\hatβ_1β^​1​是β0β_0β0​,β1β_1β1​的无偏估计

var(β^0)=var(Yˉ)=σ2/nvar(\hatβ_0)=var(\bar{Y})=σ^2/nvar(β^​0​)=var(Yˉ)=σ2/n

var(β^1)=var(∑(Xi−X^)YiSX2)=∑((Xi−Xˉ)SX2)2var(Yi)=var(Yi)/SX2=σ2/SX2var(\hatβ_1)=var(\frac{∑(X_i-\hat{X})Y_i}{{S_X}^2})\\=∑(\frac{(X_i−\bar{X})}{{S_X}^2})^2var(Y_i)\\=var(Y_i)/{S_X}^2=σ^2/{S_X}^2var(β^​1​)=var(SX​2∑(Xi​−X^)Yi​​)=∑(SX​2(Xi​−Xˉ)​)2var(Yi​)=var(Yi​)/SX​2=σ2/SX​2

  • 截距β0β_0β0​的方差由测量数据的样本决定
  • β1β_1β1​的方差由Sx2{S_x}^2Sx​2决定，X的点取得越紧密，方差越大，应尽量取分散的点
  • 可以证明，β0β_0β0​和β1β_1β1​的协方差cov(β0,β1)=0cov(β_0,β_1)=0cov(β0​,β1​)=0

Y=β0+β1(X−Xˉ)Y=β_0+β_1(X−\bar{X})Y=β0​+β1​(X−Xˉ)

b0=β0−β1Xˉb_0=β_0−β_1\bar{X}b0​=β0​−β1​Xˉ

b1=β1b_1=β_1b1​=β1​

var(b0)=var(β0)+Xˉ2var(β1)var(b_0)=var(β_0)+\bar{X}^2var(β_1)var(b0​)=var(β0​)+Xˉ2var(β1​)

var(b1)=var(β1)var(b_1)=var(β_1)var(b1​)=var(β1​)

残差与σ2的估计​

参数估计的Y与测量值Y的差，

δi=Yi−Y^iδ_i=Y_i−\hat Y_iδi​=Yi​−Y^i​

称为残差。是最小二乘法中的分量

  • 残差用于检验回归函数与数据的符合程度，不随x有特定趋势
  • 残差用于估计σ2σ^2σ2

Y=b0+b1X+eY=b_0+b_1X+eY=b0​+b1​X+e

var(Y)=var(e)=σ2var(Y)=var(e)=σ^2var(Y)=var(e)=σ2

  • 如何估计σ2σ^2σ2?

σ2=∑δi2/(n−2)σ^2=∑{δ_i}^2/(n−2)σ2=∑δi​2/(n−2)

  • 当eie_iei​符合正态分布N(0,σ2)N(0,σ^2)N(0,σ2)时，σ^2/σ2∼χn−22{\hat σ}^2/σ^2∼χ_{n-2}^2σ^2/σ2∼χn−22​
    • 可以用于估计拟合与实验数据的符合程度
import scipy.stats as st
import numpy as np
import seaborn as sns
import matplotlib.pyplot as plt

a=3
b=4
x=st.uniform(0,10).rvs(20) #生成[0,10]的20个随机样本
y=[np.random.normal(a*i+b,5) for i in x]

sns.regplot(x=x, y=y, ci=95) #画出回归线和95%的置信区间

from scipy.stats import linregress
#手动计算斜率和截距
#sx=sum (x-xi)^2
sx=np.var(x)*len(x)
sy=np.var(y)*len(y)
meanx=np.mean(x)
meany=np.mean(y)
#b1=beta1
b1= sum((x-meanx)*y)/sx
#b0=beta0-b1*xbar=ybar-b1*xbar
b0= meany-b1*meanx
residual=sum((y-b0-b1*x)**2)
varb1 = residual/sx/(len(x)-2)
varb0 = residual/len(x)/(len(x)-2)+meanx**2*varb1
print("Calculation: slope=",b1,"err=",np.sqrt(varb1)) #斜率和斜率的标准误差
print("Calculation: intercept=",b0,"err=",np.sqrt(varb0)) #截距和截距的标准误差
result=linregress(x,y) #使用scipy的linregress来验证结果
print("Fit: slope=", result.slope,result.stderr)
print("Fit: intercept=",result.intercept,result.intercept_stderr)

多元线性回归​

  • 若Y由多个因素控制

Y=b0+b1X1+b2X2⋯bpXp+eY=b_0+b_1X_1+b_2X_2⋯b_pX_p+eY=b0​+b1​X1​+b2​X2​⋯bp​Xp​+e

  • bkb_kbk​称为Y对xkx_kxk​的回归系数
  • 测量了一组n个数据

(yi,x1i,x2i,⋯,xpi),i=1,2⋯n(y_i,x_{1i},x_{2i},⋯,x_{pi}),i=1,2⋯n(yi​,x1i​,x2i​,⋯,xpi​),i=1,2⋯n

  • 写为β0β_0β0​,⋯,β1β_1β1​的中心化形式

Y=β0+β1X1∗+β2X2∗⋯bpXp∗+eY=β_0+β_1X_1^*+β_2X_2^∗⋯b_pX_p^∗+eY=β0​+β1​X1∗​+β2​X2∗​⋯bp​Xp∗​+e

  • 其中xi∗=xi−xˉx_i^∗=x_i−\bar xxi∗​=xi​−xˉ

∑j=1nxij∗=0\sum_{j=1}^{n}x_{ij}^*=0∑j=1n​xij∗​=0

bi=βi,(i=1⋯p)b_i=β_i,(i=1⋯p)bi​=βi​,(i=1⋯p)

b0=β0+∑i=1pβixib_0=β_0+\sum_{i=1}^{p}β_ix_ib0​=β0​+∑i=1p​βi​xi​

  • 为表示方便，用xxx指代x∗x^*x∗

多元线性函数的矩阵表示​

  • 用列向量来表示βββ和YYY的测量值

β=(β0⋮βp),Y=(y1⋮yn)β=\begin{pmatrix}β_0\\⋮\\β_p\end{pmatrix},Y=\begin{pmatrix}y_1\\⋮\\y_n\end{pmatrix}β=⎝⎛​β0​⋮βp​​⎠⎞​,Y=⎝⎛​y1​⋮yn​​⎠⎞​

  • n为数据的个数，p为x（系数）的个数
  • 用矩阵来表示X的测量值

X=(1 x11⋯x1p⋮⋯⋮1 xn1⋯xnp)X=\begin{pmatrix}1\ x_{11}⋯x_{1p}\\⋮⋯⋮\\1\ x_{n1}⋯x_{np}\end{pmatrix}X=⎝⎛​1 x11​⋯x1p​⋮⋯⋮1 xn1​⋯xnp​​⎠⎞​

δ=(y1−β0−x11β1−⋯−x1pβp⋮yn−β0−xn1β1−⋯−xnpβp)δ=\begin{pmatrix}y_1−β_0−x_{11}β_1−⋯−x_{1p}β_p\\⋮\\y_n−β_0−x_{n1}β_1−⋯−x_{np}β_p\end{pmatrix}δ=⎝⎛​y1​−β0​−x11​β1​−⋯−x1p​βp​⋮yn​−β0​−xn1​β1​−⋯−xnp​βp​​⎠⎞​

δ2=(Y−Xβ)⊺(Y−Xβ)δ^2=(Y−Xβ)^⊺(Y−Xβ)δ2=(Y−Xβ)⊺(Y−Xβ)

最小二乘法求解​

δ2=(Y−Xβ)⊺(Y−Xβ)δ^2=(Y−Xβ)^⊺(Y−Xβ)δ2=(Y−Xβ)⊺(Y−Xβ)

∂δ2∂β=0\frac{∂δ^2}{∂β}=0∂β∂δ2​=0

  • 矩阵求导规则, x为列向量：

∂xTa∂x=∂aTx∂x=a\frac{∂x^Ta}{∂x}=\frac{∂a^Tx}{∂x}=a∂x∂xTa​=∂x∂aTx​=a

∂xTAx∂x=Ax+ATx\frac{∂x^TAx}{∂x}=Ax+A^Tx∂x∂xTAx​=Ax+ATx

  • 带入公式

∂(Y−Xβ)⊺(Y−Xβ)∂β=∂(YT(Y−Xβ)−βTXT(Y−Xβ))∂β=XTY−XTY+(XTX+XTX)β=0\frac{∂(Y−Xβ)^⊺(Y−Xβ)}{∂β}\\=\frac{∂(Y^T(Y−Xβ)−β^TX^T(Y−Xβ))}{∂β}\\=X^TY−X^TY+(X^TX+X^TX)β=0∂β∂(Y−Xβ)⊺(Y−Xβ)​=∂β∂(YT(Y−Xβ)−βTXT(Y−Xβ))​=XTY−XTY+(XTX+XTX)β=0

  • 得到

XTXβ=XTYX^TXβ=X^TYXTXβ=XTY

β=(XTX)−1XTYβ=(X^TX)^{−1}X^TYβ=(XTX)−1XTY

令L=(XTX)L=(X^TX)L=(XTX), σ2L−1σ^2L^{−1}σ2L−1为β的协方差矩阵，且有

  • Var(β0)=σ2/nVar(β_0)=σ^2/nVar(β0​)=σ2/n
  • Var(βi)=σ2Lii−1Var(β_i)=σ^2L_{ii}^{−1}Var(βi​)=σ2Lii−1​
  • Var(β0,βj)=0Var(β_0,β_j)=0Var(β0​,βj​)=0
  • Var(βi,βj)=σ2Lij−1Var(β_i,β_j)=σ^2L_{ij}^{−1}Var(βi​,βj​)=σ2Lij−1​
  • σ2\sigma^2σ2的估计仍然可用残差来估计

δ2=(Y−Xβ)⊺(Y−Xβ)δ^2=(Y−Xβ)^⊺(Y−Xβ)δ2=(Y−Xβ)⊺(Y−Xβ)

σ2=δ2/(n−p−1)σ^2=δ^2/(n−p−1)σ2=δ2/(n−p−1)

δ2/σ2∼χn−p−12δ^2/σ^2∼χ_{n−p−1}^2δ2/σ2∼χn−p−12​

βj的区间估计​

(β^j−βj)/(σ^Ljj−1)∼tn−p−1(\hat β_j−β_j)/(\hat σ\sqrt{L_{jj}^{−1}})∼t_{n−p−1}(β^​j​−βj​)/(σ^Ljj−1​​)∼tn−p−1​

import scipy.stats as st
import numpy as np
import seaborn as sns
import matplotlib.pyplot as plt
from scipy.stats import linregress
from matplotlib.animation import FuncAnimation

np.random.seed(304)
a=4
b=4
c=10

x=st.uniform(0,10).rvs(20)
y=st.uniform(0,8).rvs(20)
z=[np.random.normal(a*i+b*j+c,5) for i,j in zip(x,y)]

将数据点用三维动图展示出来

def animate(dfn):
    ax.view_init(elev=32, azim=dfn)
    
fig = plt.figure(figsize=(4, 4))

ax = fig.add_subplot(111, projection='3d')
ax.plot(x, y, z, color='k', zorder=15, linestyle='none', marker='o', alpha=0.5)

anim = FuncAnimation(fig, animate, frames=range(0,360,2))
anim.save('data.gif',fps=3,dpi=200)

fig.show()

矩阵运算求线性回归系数​

按照公式手动求解

one=np.ones(20)
Xm=np.vstack([one,x,y]).T
Y = np.array(z).reshape(20,1)
b=np.linalg.solve(Xm.T@Xm,Xm.T@Y)
print(b)

矩阵运算求线性回归误差​

手动求解

Xmm=np.vstack([one,x-np.mean(x),y-np.mean(y)]).T
beta=np.linalg.solve(Xmm.T@Xmm,Xmm.T@Y)
print(beta,"\n")
L_inv=np.linalg.inv(Xmm.T@Xmm)
sigma2=(Y-Xmm@beta).T @(Y-Xmm@beta)/(20-2-1)
print(L_inv,"\n")
err=np.sqrt(sigma2*np.diag(L_inv))
print(err)
